HTML Básico

Por sus siglas en inglés HyperText Markup Language (Lenguaje de marcado para hipertextos). Es el elemento de construcción más básico de una página web y se usa para crear y representar visualmente la misma. Es importante especificar que HTML determina el contenido de la página web, pero no su funcionalidad.

Elementos básicos (Tags)

  • Comentarios
  • (*)!DOCTYPE html: identificador de versión.
  • (*)HTML: indica que todo lo que está dentro de este bloque se debe interpretar como HTML
    • (*)HEAD: la cabecera contiene información propia del documento
      • META: contenedor de metainformación sobre el documento
      • (*)TITLE: título del docuemnto
      • LINK: describe las relaciones del documento
    • (*)BODY: engloba el conetenido del documento
      • SECTION: representa una sección genérica de un documento.
      • NAV: sección con contenido de navegación.
      • ARTICLE: especifica el contenido independiente, autónomo.
      • HEADER: cabecera del docuemnto, suele contener elementos de navegación y encabezados.
      • FOOTER: pie del documento
      • ASIDE: cualquier elemento adicional, al margen del flujo principal del texto.
