CSS

Hojas de Estilo en Cascada (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. CSS describe como debe ser renderizado el elemento estructurado en pantalla.

Regresar

Existen tres formas de integrar CSS

  • En línea
  • En head o cabecera
  • En hoja de estilos externa

Etiqueta img

El elemento de imagen HTML img representa una imagen en el documento, sus atributos son:

  • src: La URL de la imagen. Este atributo es obligatorio para la etiqueta.
  • alt: define el texto alternativo que describe la imagen.
